# 🧮 Calculadora Científica en Java

## 📌 Descripción

Este proyecto consiste en el desarrollo de una calculadora científica en Java, diseñada para estudiantes de ingeniería. Permite realizar operaciones matemáticas básicas y avanzadas, además de guardar un historial de los cálculos realizados durante la ejecución del programa.

---

## ⚙️ Funcionalidades

La calculadora permite realizar las siguientes operaciones:

* ➕ Suma
* ➖ Resta
* ✖️ Multiplicación
* ➗ División (con validación de división por cero)
* √ Raíz cuadrada
* ^ Potencia
* sen() Seno
* cos() Coseno

También incluye:

* 📜 Historial de operaciones realizadas
* 🔢 Resultado actualizado en cada cálculo

---

## 🛠️ Tecnologías utilizadas

* Java
* Librerías estándar:

  * `java.util.Scanner`
  * `java.util.ArrayList`

---

## 🚀 Ejecución del programa

1. Compilar el archivo:

```
javac Main.java
```

2. Ejecutar el programa:

```
java Main
```

---

## 🧠 Uso

1. El usuario selecciona una opción del menú.
2. Ingresa los valores solicitados.
3. El programa muestra el resultado.
4. Cada operación se guarda en el historial.

Ejemplo:

```
SUMA (a + b)
Ingresa número 1: 5
Ingresa número 2: 3
Resultado: 8.0
```
---
## 📂 Estructura del proyecto

```
Main.java
```

Todo el programa se encuentra en un único archivo para facilitar su ejecución.

---

## 📌 Notas importantes

* Las funciones seno y coseno trabajan en **radianes**.
* No se permite la división por cero.
* El historial se mantiene solo durante la ejecución del programa.
